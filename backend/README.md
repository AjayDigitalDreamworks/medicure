

For Security :- 


1. Helmet  -   HTTP headers ke zariye app ko secure banata hai

2. morgan  -   Logging ke liye use hota hai, mainly HTTP request logs

3. express-rate-limit   -    Rate limiting — repeated requests ko block karta hai.

4. crypto                -     Encryption & hashing

5. passportjs             -   Authentication middleware

6. express-mongo-sanitize   - MongoDB ke query injection attacks se bachaata hai

7. xss-clean                 -   User input se malicious HTML/JS content ko clean karta hai

8. zod               -   Schema validation for user input  ( Input validation — malicious ya invalid data ko reject karta hai )

9. express-session   -   Session management.

10. bcrypt            -   Password hashing.

11. cors         -    Cross-Origin Resource Sharing policies define karta hai ( domains block kerta hai jo unauthorized ho )
